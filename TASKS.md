## 🛠️ 快速开始

### 开发环境设置
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建扩展
npm run build

# 类型检查
npm run build:check
```

### Chrome扩展安装
1. 运行 `npm run build` 构建项目
2. 打开Chrome扩展管理页面 `chrome://extensions/`
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `dist` 文件夹
